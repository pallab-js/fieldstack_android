# Changelog

## [0.1.0-mvp] — 2026-05-01

### Added
- Offline-first task management (view, filter, search, complete)
- 5-step report builder (basic info, details, photos, location, signature)
- CameraX photo capture + FusedLocation attachment
- Barcode/QR scanning via ML Kit (auto-fills asset ID)
- WorkManager sync engine with exponential backoff + delta sync protocol
- Conflict resolution: last-write-wins strategy
- BudgetZen design system (Mint/Sky/WarmGray, Manrope/Nunito fonts)
- Role-based access control (FieldTech / Supervisor / Admin)
- Task comments with @mention Sky-blue highlighting
- Weekly insights dashboard + CSV export
- PDF report export (Android PdfDocument API)
- Custom form fields (Text / Number / Checkbox / Date)
- Biometric unlock (BiometricPrompt)
- Full TalkBack accessibility (WCAG AA contrast, heading semantics)
- GitHub Actions CI/CD (tests on PR, signed release on tag push)
- Firebase App Distribution for internal QA
