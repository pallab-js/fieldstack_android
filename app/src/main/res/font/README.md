# BudgetZen Fonts

Download these font files from Google Fonts and place them here:

| File name                    | Google Fonts URL                                      |
|------------------------------|-------------------------------------------------------|
| manrope_regular.ttf          | https://fonts.google.com/specimen/Manrope             |
| manrope_semibold.ttf         | https://fonts.google.com/specimen/Manrope             |
| manrope_bold.ttf             | https://fonts.google.com/specimen/Manrope             |
| manrope_extrabold.ttf        | https://fonts.google.com/specimen/Manrope             |
| nunito_regular.ttf           | https://fonts.google.com/specimen/Nunito              |
| nunito_semibold.ttf          | https://fonts.google.com/specimen/Nunito              |
| source_code_pro_regular.ttf  | https://fonts.google.com/specimen/Source+Code+Pro     |

## Quick download (macOS terminal)

```bash
cd app/src/main/res/font

# Manrope
curl -L "https://github.com/sharanda/manrope/raw/master/fonts/otf/Manrope-Regular.otf" -o manrope_regular.ttf
curl -L "https://github.com/sharanda/manrope/raw/master/fonts/otf/Manrope-SemiBold.otf" -o manrope_semibold.ttf
curl -L "https://github.com/sharanda/manrope/raw/master/fonts/otf/Manrope-Bold.otf" -o manrope_bold.ttf
curl -L "https://github.com/sharanda/manrope/raw/master/fonts/otf/Manrope-ExtraBold.otf" -o manrope_extrabold.ttf
```

Alternatively, use Android Studio → res/font → right-click → "More" → "Downloadable Fonts"
and search for each family.
